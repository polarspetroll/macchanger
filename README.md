# macchanger
mac address changer

## futures
**+** built in small database with popular hardware authors

**+** light weight 

**+** easy to use and install


### installation 

**quick usage**


``
git clone https://github.com/polarspetroll/macchanger.git

chmod +x mchanger.rb

./mchanger.rb
``

**full install**

``
chmod +x install.sh

./install.sh

mchanger.rb
``
